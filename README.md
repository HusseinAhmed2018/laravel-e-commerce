## About Project

This is Laravel simple E-commerce project using: 
- vue js component 
- Datatable ajax request
- Laravel pagination
- Seeding Data (2 type of user, upto 50 products, 1 store)
- custom install command to run migrate and seed date

## Project Install
- composer install
- php artisan project:install
